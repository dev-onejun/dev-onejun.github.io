## ppa

### deadsnakes

The latest python version in ubuntu.

``` bash
sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt update
```

### Jonathon F

The latest vim version in ubuntu.

``` bash
sudo add-apt-repository ppa:jonathonf/vim
sudo apt update
```
